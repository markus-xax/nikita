# Примеры по документации

Здесь собраны практические примеры для изучения ванильного фронтенда.

## Структура примеров

### Пример 1: Базовый HTML (`01-basic-html.html`)
**Что изучаем:**
- Основная структура HTML документа
- Семантические теги (header, nav, main, section, footer)
- Заголовки, параграфы, списки
- Ссылки и изображения
- Подключение CSS файла

**Как использовать:**
1. Откройте `01-basic-html.html` в браузере
2. Изучите исходный код
3. Попробуйте изменить содержимое

---

### Пример 2: Формы (`02-forms.html`)
**Что изучаем:**
- Различные типы input элементов
- Работа с формами (form, input, select, textarea)
- Обработка событий формы в JavaScript
- Валидация данных
- Отображение результатов

**Файлы:**
- `02-forms.html` - HTML разметка формы
- `styles-2.css` - стили для формы
- `script-2.js` - обработка формы в JavaScript

**Как использовать:**
1. Откройте `02-forms.html` в браузере
2. Заполните форму и отправьте
3. Изучите код обработки формы в `script-2.js`

---

### Пример 3: Интерактивная страница (`03-interactive-page.html`)
**Что изучаем:**
- Работа с событиями (click, submit, keypress)
- Изменение состояния элементов
- Создание динамического контента
- Работа с модальными окнами
- Переключение тем оформления
- Создание списка задач (TODO)

**Файлы:**
- `03-interactive-page.html` - HTML разметка
- `styles-3.css` - стили с темами
- `script-3.js` - вся интерактивность

**Как использовать:**
1. Откройте `03-interactive-page.html` в браузере
2. Попробуйте все интерактивные элементы:
   - Счетчик (увеличить/уменьшить/сбросить)
   - Добавление задач в список
   - Переключение тем
   - Открытие модального окна
3. Изучите код в `script-3.js` для понимания работы

---

### Пример 4: Flexbox и Grid (`04-flexbox-grid.html`)
**Что изучаем:**
- CSS Flexbox для выравнивания элементов
- CSS Grid для создания сеток
- Различные конфигурации Flexbox (row, column, space-between)
- Grid с разными размерами элементов
- Практический пример - карточки товаров
- Адаптивный дизайн

**Файлы:**
- `04-flexbox-grid.html` - HTML с примерами
- `styles-4.css` - все стили Flexbox и Grid

**Как использовать:**
1. Откройте `04-flexbox-grid.html` в браузере
2. Изучите различные примеры Flexbox и Grid
3. Попробуйте изменить значения в CSS файле
4. Измените размер окна браузера, чтобы увидеть адаптивность

---

### Пример 5: Манипуляции с DOM (`05-dom-manipulation.html`)
**Что изучаем:**
- Создание новых элементов (`createElement`)
- Добавление элементов в DOM (`appendChild`)
- Изменение стилей через JavaScript (`style`)
- Работа с классами (`classList.add/remove/toggle`)
- Работа с атрибутами (`setAttribute`, `getAttribute`)
- Изменение содержимого (`textContent`, `innerHTML`)
- Удаление элементов (`remove`)

**Файлы:**
- `05-dom-manipulation.html` - HTML разметка
- `styles-5.css` - стили
- `script-5.js` - все манипуляции с DOM

**Как использовать:**
1. Откройте `05-dom-manipulation.html` в браузере
2. Попробуйте все кнопки и функции
3. Изучите код в `script-5.js` - это основа работы с DOM

---

## Порядок изучения

Рекомендуемый порядок для начинающих:

1. **Пример 1** - начните с базового HTML, чтобы понять структуру
2. **Пример 2** - изучите формы, это важная часть веб-разработки
3. **Пример 5** - изучите манипуляции с DOM, это база для интерактивности
4. **Пример 3** - примените знания DOM для создания интерактивной страницы
5. **Пример 4** - изучите современные методы верстки (Flexbox и Grid)

## Как работать с примерами

1. **Откройте HTML файл в браузере** - просто дважды кликните или откройте через браузер
2. **Изучите код** - откройте файлы в редакторе кода
3. **Экспериментируйте** - изменяйте код и смотрите, что происходит
4. **Повторяйте** - попробуйте создать свои версии примеров

## Советы

- Используйте консоль разработчика браузера (F12) для отладки
- Читайте комментарии в коде - они объясняют, что происходит
- Не бойтесь экспериментировать и делать ошибки
- Создавайте свои примеры на основе изученного материала

## Следующие шаги

После изучения всех примеров попробуйте:
- Создать свой проект с нуля
- Добавить новые функции к существующим примерам
- Изучить более продвинутые темы (API, асинхронность, модули)
